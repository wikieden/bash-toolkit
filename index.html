bash-toolkit by Zealot Ke

Bash Toolkit - Bash工具箱
see https://github.com/chzealot/bash-toolkit

概述 | Overview

提高Bash使用效率的配置文件和工具

configuraion and tools for developers in Bash for efficiency.

特点 | Features

  1. Bash使用更加方便，例如grep命令带颜色、忽略.svn目录
  2. 常用命令支持终端颜色，例如svn、gcc、make和diff
  3. 支持z工具，记录并自动在目录间跳转，基于使用频率(z foobar, z foo, etc.)
  4. 支持bashmarks，设置目录书签，跳转到书签对应的目录(s设置,g跳转,l列出书签列表)
  5. 支持ack工具，类似grep但更好用，更强大
  6. 支持动态motd，自定义登陆shell时欢迎消息(/etc/motd为静态文本信息)
  7. 平台自适应，自动加载系统对应的特定配置，例如Mac、Linux等
  8. 用户名自适应，自动根据用户名加载对应配置
  9. Mac下优化，例如中文显示、Mac中特殊路径
  1. Make it easier to get comfortable in Bash, such as "to ignore .svn directory for 'grep'"
  2. Support console colorize, like svn,gcc,make and diff
  3. Support z, Tracks your most used directories, based on 'frecency'
  4. Support bashmarks, Directory bookmarks for the shell
  5. Support ack, better than "grep"
  6. Support dynamic motd(/etc/motd)
  7. Platform self-adapting, load difference config in difference system automatically, like Mac, Linux, etc
  8. Username self-adapting, load difference config for difference user automatically
  9. Support chinese character friendly for the shell in Mac

依赖 | Dependencies

bash

安装 | Install

  1. $ wget --no-check-certificate "https://github.com/chzealot/bash-toolkit/raw/master/install.sh" -O install.sh
  2. $ chmod +x install.sh
  3. $ ./install.sh
  4. append the follow code snippets into ~/.bashrc or "~/.bash_profile" 
    if [ -f ~/.bash.d/bashrc ]; then
    . ~/.bash.d/bashrc
fi
